功能描述

点击【退出】，通过调用接口omeng.close()，将本画面关闭

点击【播放网络视频】，通过调用接口omeng.openVideo(url)将视频地址传递给后台，视频播放器打开

代码如下

<a href="javascript:void(0);" onclick="omeng.close();">退出</a><br/>
<a href="javascript:void(0);" onclick="openVideo();">播放网络视频</a><br/>
<script type="text/javascript">
function openVideo() {
    omeng.openVideo("http://jlpt.img-cn-hangzhou.aliyuncs.com/cloud/ico/test/2017-03-01/1488359719148_dAA3KM6h.mp4");
}
</script>
